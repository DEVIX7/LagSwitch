# LagSwitch

This program was created in python and allows you to disable / enable the Internet by pressing keys

> Version: `1.5`

### Hot keys
* Ctrl + E = Internet enabled
* Ctrl + Q = Internet disabled
* Shift + Esc = Close Lag switch
